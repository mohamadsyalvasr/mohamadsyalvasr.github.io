Panduan Let's Encrypt SSL

Aplikasi ini membantu Anda membuat perintah Certbot yang sesuai dengan konfigurasi server Anda.

Langkah-langkah untuk Server Anda

Berikut adalah perintah yang dapat Anda gunakan di server Anda berdasarkan pilihan Anda:

Perintah Certbot

Uji Perpanjangan Otomatis

Anda dapat menguji perpanjangan otomatis menggunakan perintah berikut: 

sudo certbot renew --dry-run

Penting: Panduan ini tidak menggantikan konfigurasi server Anda. Sesuaikan dengan kondisi di server masing-masing.
