Perbandingan Cloud Compute

Perbandingan spesifikasi Virtual Machine (VM) head-to-head antara AWS, Microsoft Azure, dan Google Cloud Platform.

Cara Membaca Tabel

Tabel di bawah ini membandingkan seri VM dari setiap provider berdasarkan kasus penggunaan umum. Perlu diingat bahwa ini adalah padanan terdekat; performa bisa bervariasi karena perbedaan arsitektur dasar, jenis penyimpanan, dan jaringan. Harga tidak disertakan karena sangat bervariasi berdasarkan wilayah, model reservasi, dan waktu.

Klik pada judul setiap kategori untuk membuka atau menutup tabel perbandingan.
